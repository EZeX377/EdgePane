Including Required Libraries

To use EdgePane on your page, include the following CDNs in this order:

  • EdgePane CSS & JS: Include your hosted EdgePane files first (or download them to your server): 
    <link rel="stylesheet" href="path/to/egp-sidebar.css">
<script src="path/to/egp-sidebar.js"></script>
  • jQuery: Required for sidebar functionality. Must be version 3.x or higher: 
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
  • CSS Framework (Optional): Any framework like TailwindCSS or Bootstrap can be included after jQuery: 
    <link href="https://cdn.jsdelivr.net/npm/tailwindcss@3.3.3/dist/tailwind.min.css" rel="stylesheet">
  • Icons (Optional): Any icon library such as Bootstrap Icons or Font Awesome: 
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.css">

Following this order ensures that the sidebar works correctly and your styles/icons render properly.
