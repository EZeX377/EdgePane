Responsive Behavior

EdgePane is fully responsive and adapts automatically to desktop, tablet, and mobile devices.

  • On mobile/tablet:
    • Sidebar is hidden by default
    • .egp-sidebar-overlay appears when sidebar is open
    • Toggling works via .egp-sidebar-toggle button
  • On desktop:
    • Sidebar state (open | closed) is respected
    • Overlay is not used, sidebar stays docked
    • Hover expand works when hoverExpand: true is set

  • Use sidebarState: "open" or "closed" in config for default desktop behavior
