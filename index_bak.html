üìò EdgePane Documentation (EGP)

EdgePane is a lightweight, responsive, and customizable sidebar component built with custom CSS and jQuery. All custom classes are now namespaced with egp- to avoid conflicts with other frameworks. Try it live here: EdgePane Demo

üìå Requirements

  • jQuery (required; load before the script)
  • Works with any CSS framework (Bootstrap used in demo)
  • Use any icon library (e.g., Bootstrap Icons, Font Awesome)

üåê CDN Links

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/EZeX377/EdgePane@main/dist/edgepane_egp.css">
<script src="https://cdn.jsdelivr.net/gh/EZeX377/EdgePane@main/dist/edgepane_egp.js"></script>

üöÄ Usage

Place a toggle button and overlay outside the sidebar, and the full sidebar markup:

<!-- Toggle Button -->
<button class="egp-sidebar-toggle">
  <i class="bi bi-list"></i>
</button>

<!-- Overlay (for small screens) -->
<div class="egp-sidebar-overlay"></div>

<!-- Sidebar -->
<nav class="egp-sidebar bg-neutral-800" role="navigation">
  <div class="egp-sidebar-header">
    <img src="./assets/images/logo-d.png" alt="Logo" class="egp-sidebar-logo" />
    <div class="egp-sidebar-title">
      <p class="egp-sidebar-brand-name">EdgePane</p>
      <p class="egp-sidebar-brand-tagline">Pane Perfect Experience</p>
    </div>
  </div>

  <ul class="egp-sidebar-menu">
    <!-- GENERAL -->
    <li class="egp-sidebar-section-heading" data-section-heading="GENERAL"></li>
    <li><a href="#" class="egp-sidebar-link"><span class="egp-sidebar-link-icon"><i class="bi bi-columns-gap"></i></span><span class="egp-sidebar-link-text">Dashboard</span></a></li>

üì± Responsive

On mobile/tablet:

  • Sidebar is hidden by default
  • .egp-sidebar-overlay is shown when sidebar is open
  • Toggling works via .egp-sidebar-toggle button

‚ùì FAQ

Sidebar doesn‚Äôt show? Include both edgepane_egp.css & edgepane_egp.js, and load jQuery before them.

Toggle not working? Ensure button has class .egp-sidebar-toggle.

Dropdowns not expanding? Structure must be .egp-sidebar-dropdown > .egp-dropdown-toggle + .egp-dropdown-content.

Active links? Last clicked link is saved in localStorage and restored on reload.
