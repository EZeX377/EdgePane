# EdgePane

A lightweight, responsive, and customizable **sidebar component** powered by jQuery.  
Easily add collapsible menus, active link highlighting, and branding to your web apps.

---

## 📦 Installation

Include jQuery and EdgePane in your project:

```html
<link rel="stylesheet" href="edgepane.css">
<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
<script src="edgepane.min.js"></script>
Or use via CDN:

html
Copy code
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/edgepane/dist/edgepane.css">
<script src="https://cdn.jsdelivr.net/npm/edgepane/dist/edgepane.min.js"></script>
