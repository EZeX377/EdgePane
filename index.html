Introduction

EdgePane is a modern, responsive sidebar for web apps and dashboards. It supports sections, dropdowns, icons, collapsible functionality, and a toggle button for mobile devices.

Requirements

  • jQuery (v3+)

EdgePane uses custom CSS and works with any CSS framework. You can use any icon library; the demo uses Bootstrap Icons.

Getting Started

1. Include Dependencies

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.5/font/bootstrap-icons.css">

2. Add Sidebar Layout

Include the sidebar HTML and the toggle button:

<button class="sidebar-toggle"><i class="bi bi-list"></i></button>

<nav class="sidebar">
  <div class="sidebar-header">
    <img src="" class="sidebar-logo" alt="Logo">
    <div class="sidebar-title">
      <p class="sidebar-brand-name"></p>
      <p class="sidebar-brand-tagline"></p>
    </div>
  </div>

  <ul class="sidebar-menu">
    <li class="sidebar-section-heading" data-section-heading="GENERAL"></li>
    <li><a href="#" class="sidebar-link">
      <span class="sidebar-link-icon"><i class="bi bi-columns-gap"></i></span>
      <span class="sidebar-link-text">Dashboard</span></a></li>
  </ul>
</nav>

3. Initialize EdgePane

Example initialization with full options:

edgePane.init({
    sidebarState: "open",
    hoverExpand: true,
    closeOnClickOutside: true,
    dropdownMode: "single",
    rememberDropdowns: true,
    fontFamily: "'Montserrat', sans-serif",
    sidebarWidth: "18rem",
    sidebarColor: "rgba(225,225,225,1)",
    textColor: "rgba(70,70,70,1)",
    activeLinkBg: "rgba(180,180,180,1)",
    activeLinkColor: "rgba(70,70,70,1)",
    showBrand: true,
    brand: {
        brandLogoSrc: "./assets/images/logo-d.png",
        brandName: "EdgePane",
        brandTagline: "Pane Perfect Experience",
    },
});

Features

  • Toggle Button: Allows opening/closing the sidebar on small screens.
  • Collapsible Sidebar: Users can toggle sidebar open/closed.
  • Dropdown Menus: Nested links with optional single/multi open behavior.
  • Sections: Automatically labeled using data-section-heading.
  • Icons: Any icon library can be used for menu icons.

Customization

HTML Customization

OptionDescription
.sidebar-logoSidebar brand/logo image.
.sidebar-brand-nameBrand name text.
.sidebar-brand-taglineOptional tagline below the brand name.
.sidebar-link-iconMenu item icon (any library).
.sidebar-link-textText for menu links.
.sidebar-dropdownWrap submenu items for collapsible dropdowns.
data-section-headingAutomatically label sections in the sidebar.

Initialization Customization

OptionDescription
sidebarStateInitial state: "open" or "closed".
hoverExpandExpand sidebar on hover when collapsed.
closeOnClickOutsideCloses the sidebar if user clicks outside.
dropdownMode"single" (only one dropdown open) or "multi".
rememberDropdownsKeep dropdowns open after page reload.
fontFamilyCustom font family for sidebar text.
sidebarWidthWidth of expanded sidebar.
sidebarColorBackground color of sidebar.
textColorColor of menu text.
activeLinkBgBackground color of active link.
activeLinkColorText color of active link.
showBrandToggle brand visibility.
brandObject containing brandLogoSrc, brandName, brandTagline.

Tips for Users

  • Include jQuery before initialization.
  • Toggle button allows easy sidebar access on small screens.
  • DropdownMode "single" or "multi" controls menu behavior.
  • Customize appearance with sidebarColor, textColor, fontFamily, etc.
  • Brand can be shown or hidden using showBrand.
